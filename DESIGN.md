# Контрейл — дизайн-система «Морской фрахт»

> Образ: **грузовой терминал, а не лендинг.** Биржа фрахта в реальном времени —
> плотная, быстрая, точная. Промышленная родословная (контейнеры, море, сталь),
> поданная с редакторской сдержанностью. Ближе к торговому терминалу и грузовой
> накладной, чем к маркетинговому сайту.
>
> Три слова-ориентира: **точность · плотность · доверие.**

---

## 1. Цвет

Прохладный бетон + чернильно-бирюзовый. Два цвета, использованные уверенно, — не радуга.

### Поверхности (прохладный бетон)
| Токен | HEX | Назначение |
|---|---|---|
| `paper`         | `#F3F5F4` | фон приложения |
| `surface`       | `#FBFCFB` | карточки, строки доски |
| `surface-sunken`| `#ECEFEE` | вложенные блоки, поля ввода в покое |
| `hairline`      | `#DDE3E1` | волосяные линии (основной разделитель) |
| `border-strong` | `#C7CFCD` | акцентная граница, hover |

### Чернила (текст, чёрно-зелёный)
| Токен | HEX | Назначение |
|---|---|---|
| `ink`   | `#10201F` | основной текст, заголовки |
| `ink-2` | `#3A4A48` | вторичный текст |
| `ink-3` | `#64748B` | подписи, оверлайны, иконки в покое (slate) |
| `ink-4` | `#94A3A0` | приглушённый/плейсхолдеры |

### Акцент (чернила-бирюза)
| Токен | HEX | Назначение |
|---|---|---|
| `accent`         | `#0E6E6E` | основное действие, активные состояния |
| `accent-hover`   | `#0B5A5A` | ховер |
| `accent-pressed` | `#084848` | нажатие |
| `accent-soft`    | `#E1EEED` | мягкая заливка (фон бейджа, hover строки) |
| `accent-ring`    | `rgba(14,110,110,.35)` | focus-ring |

### Семантика (приглушённая, без неона)
| Статус | Текст/точка | Мягкий фон | Где |
|---|---|---|---|
| success | `#2E6B4F` хвоя | `#E4EFEA` | доставлено, принято |
| warning | `#B4671C` янтарь-кирпич | `#F6ECDD` | срочно, ожидает |
| danger  | `#A93B32` кирпич | `#F5E3E1` | отмена, ошибка |
| neutral | `ink-3` slate | `surface-sunken` | закрыто, черновик |

**Статусы заявки:** `active` → accent · `matched` → slate · `in_transit` → warning ·
`delivered` → success · `cancelled/closed` → neutral · маркер «срочно» → кирпичная точка.

## 2. Типографика

Уходим от дефолтного Inter (тель №1 «ИИшности»). Обе гарнитуры — с полной кириллицей.

- **UI:** `Onest` (400 / 500 / 600 / 700) — через `next/font`.
- **Данные:** `Martian Mono` (400 / 500) — цены, вес, даты, номера заявок и контейнеров.
  Всегда `font-variant-numeric: tabular-nums`, выравнивание в колонку.

### Шкала
| Роль | Размер / интерлиньяж / трекинг | Вес | Гарнитура |
|---|---|---|---|
| display (герой) | 48 / 1.05 / −0.02em | 700 | Onest |
| h1 | 32 / 1.1 / −0.015em | 700 | Onest |
| h2 | 24 / 1.15 / −0.01em | 600 | Onest |
| h3 | 18 / 1.25 | 600 | Onest |
| body-lg | 17 / 1.5 | 400 | Onest |
| body | 15 / 1.5 | 400 | Onest |
| small (подпись) | 13 / 1.45 | 500 | Onest |
| overline (метка/бейдж) | 11.5 / 0.06em / UPPERCASE | 600 | Onest |
| data (цена/код) | 15 / tabular | 500 | Martian Mono |
| data-sm (дата/мета) | 13 / tabular | 400 | Martian Mono |

## 3. Форма, отступы, глубина

- **Сетка:** база 4px. Шаг 4 / 8 / 12 / 16 / 20 / 24 / 32 / 40 / 56 / 72.
- **Радиусы (сдержанно, НЕ 2xl везде):** `sm 6px` (поля, бейджи) · `md 10px` (карточки,
  кнопки) · `lg 14px` (модалки). Полное скругление (pill) — только у точек-статусов и тегов.
- **Границы vs тени:** по умолчанию — **волосяная граница 1px `hairline`**, а не тень.
  Единственная тень-токен — для оверлеев/меню: `0 8px 24px -8px rgba(16,32,31,.18)`.
  Карточка = граница, не `shadow-lg`.
- **Плотность — два режима:** маркетинг дышит; рабочие экраны плотные (высота строки
  доски ~52–56px, `py-3`, липкие шапки, волосяные разделители).

## 4. Компоненты

- **Button.** primary: заливка `accent`, белый текст, hover `accent-hover`, `radius-md`,
  min-h 44, вес 500. secondary: `surface` + `hairline`, текст `ink`. ghost: прозрачный,
  `ink-3`→`ink`. danger: кирпич. Focus — `accent-ring`. В кнопках моно не используем.
- **Строка доски (лента).** Разделены волосяной линией на `surface`. Слева — маршрут
  (города Onest 15/600, коннектор `──•──` пунктиром-«рельсой», узел via — кружок с
  accent-кольцом). Бейдж контейнера — bordered-chip, overline mono. Цена — Martian Mono 15
  tabular. Мета (дата) — data-sm `ink-3`. Срочно — кирпичная точка + `СРОЧНО` overline.
  Hover: фон `accent-soft` + левая полоса 2px `accent`. Новая строка (realtime) — краткая
  вспышка-подсветка accent→прозрачность 1.2s.
- **Карточка (деталь).** `surface`, `hairline`, `radius-md`, паддинг 20–24. Секции —
  overline-метки `ink-3`.
- **Status pill.** мягкий фон + тонированный текст, `radius-sm`, overline 11.5 UPPERCASE,
  крошечная ведущая точка. Без насыщенных заливок.
- **Input.** `surface`, `hairline`, focus = граница `accent` + `accent-ring`. Метка —
  overline. Числовые поля (цена, ИНН) — Martian Mono. Высота 44.
- **Тег / тип контейнера.** bordered-chip, mono UPPERCASE 11.5, `ink-2` на `surface-sunken`.
- **Маршрут.** `Откуда ──•── Куда`, 1px пунктир-«рельса», узел via — accent-кольцо;
  иконку-стрелку убираем.
- **Пустые состояния.** по центру, сдержанная линейная иконка, копия `ink-3`, одно
  accent-действие. Без эмодзи.
- **Загрузка.** скелетон-строки под доску (мягкий shimmer), НЕ спиннер. Спиннер — только
  инлайн в кнопке.
- **Навигация.** `surface`, нижняя `hairline`, вордмарк слева, активный пункт = текст
  `accent` + подчёркивание 2px (не залитый pill). Плотнее текущей.

---

## 5. Иконки, бренд, движение

- **Иконки:** lucide, обводка 1.5, размер 16–18, в покое `ink-3`. Заменить эмодзи-бейджи
  (🚚 / ⏸) на lucide (`Truck`, `PauseCircle`), тонируя по статусу.
- **Вордмарк:** «Контрейл» набран Onest 700, −0.02em, `ink`; рядом собственный мини-знак —
  глиф контейнера/узла-маршрута в `accent` (нарисуем SVG). Он же favicon и OG-превью.
- **Движение:** 120–180ms, `cubic-bezier(.2,.6,.2,1)`. Только фон/линия на ховере.
  Realtime-вспышка новой строки 1.2s ease-out. Без fade-up и пружин.

---

## 6. Как меняются экраны

- **Лендинг.** Прохладная бумага, крупный Onest-display. Герой-визуал = **живой мок самой
  доски** (терминал как герой, а не абстрактная иллюстрация). Полоса доверия
  (города / маршруты / заявки), секции с волосяными разделителями, бирюзовый CTA.
  Текстовые `✓` → lucide-check в accent.
- **Лента / доска.** Из грид-карточек → плотная терминальная доска с липкой строкой
  фильтров, моно-колонками и realtime-вспышкой.
- **Дашборд.** Тот же язык доски для «моих заявок» + компактная шапка со статистикой.
- **Деталь заявки/машины.** Редакторские две колонки, герой-маршрут, моно-таблица данных,
  status-pill.
- **Чат.** Пузыри перекрашиваем в `ink`/`accent`, волосяные границы.
- **Профиль / формы.** Overline-метки, числовые поля моноширинным.

---

## 7. Анти-ИИ чек-лист (чего НЕ делаем)

Фиолетовые градиенты и глассморфизм · размытые «блобы» · всё `rounded-2xl` + тень ·
дефолтный Inter/Geist · эмодзи как UI · центрированный герой с градиентом ·
кислотные статусы · одинаковый отступ 24px везде без вариаций плотности.
